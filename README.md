# B站直播流抓取工具

ROOMID查看方式：进入B站直播间 -> 右键 -> 查看源代码 -> 第24行

## 技术栈
pug + sass + coffeescript + vue + bootstrap + webpack + nwjs。

## 目录结构
* nwjs: nwjs SDK
  * app: 源代码
  * dependent: 依赖的文件存储目录
    * ffmpeg: ffmpeg
  * output: 视频输出目录
